# Initial Release Notes

## Advanced NSE Portfolio Optimizer

Initial GitHub-ready release of a Python and Streamlit based portfolio optimization dashboard for Indian NSE stocks.

### Included

- Modular Python project structure
- Streamlit dashboard entrypoint at `app.py`
- NSE ticker configuration in `config.py`
- Data loading and cleaning utilities
- Portfolio return and risk calculations
- SciPy optimization for Maximum Sharpe and Minimum Volatility portfolios
- Monte Carlo simulation
- Risk analytics with drawdown, Sortino Ratio, VaR, and CVaR
- Visualization helpers with pure black dashboard theme
- README, deployment instructions, LinkedIn post, and resume bullets
