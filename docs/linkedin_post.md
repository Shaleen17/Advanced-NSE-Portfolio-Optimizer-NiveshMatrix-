# LinkedIn Project Announcement Post

I’m excited to share my project: **Advanced NSE Portfolio Optimizer**.

This is a Python and Streamlit based dashboard for analyzing and optimizing Indian NSE stock portfolios using quantitative finance concepts.

What the project includes:

- NSE stock data collection using `yfinance`
- Adjusted Close price based return analysis
- Daily returns, annual returns, volatility, covariance, and correlation
- Random portfolio generation
- Monte Carlo simulation
- Efficient Frontier style visualization
- Maximum Sharpe Ratio optimization
- Minimum Volatility optimization
- SciPy constrained optimization
- Backtesting-oriented risk analysis
- Advanced risk metrics such as Sortino Ratio, Maximum Drawdown, VaR, and CVaR
- Black-Litterman, factor investing, and machine learning experimentation
- Streamlit dashboard with a black, white, green, and red fintech theme

The goal of this project was to connect finance theory with practical Python implementation and build something suitable for both academic submission and GitHub portfolio presentation.

Important note: This project is for educational purposes only and does not provide financial advice or guarantee investment returns.

Tech stack: Python, Streamlit, pandas, NumPy, SciPy, yfinance, Matplotlib, Seaborn, scikit-learn.
